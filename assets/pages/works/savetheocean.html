Save The Ocean - 2024

Projeto de Global Solution educativo com proposta feita pela universidade FIAP, cuja o objetivo é sanar um problema global. Onde neste caso o tema é a vida nos oceanos.

O sistema criado tem o objetivo de aproximar o cidadão comum as ONG's e empresas que trabalham diariamente para salvar os oceanos do planeta. Sendo um aplicativo simples, rápido e intuitivo, com o propósito de ser um canal de doações prático e direto.

Toda a identidade visual foi criada do zero, desde o logo, cores e o design do sistema. Por se tratar de um projeto educativo, foi utilizado apenas Figma, HTML, CSS, JavaScript e Bootstrap para chegar no resultado final.
